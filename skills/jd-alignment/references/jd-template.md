# 典型 Java 后端 JD 示例

## 岗位要求
1. 了解LLM服务、Agent应用、MCP/A2A和Python优先；
2. 3年以上后端开发经验，掌握但不限于JAVA/Golang/Python/C/C++等编程语言中的一种或几种；
3. 熟悉网络编程和多线程编程，对TCP/IP/HTTP等网络协议有一定的理解；
4. 熟悉数据库技术，熟练编写SQL脚本，熟悉Redis、MongoDB、Elasticsearch等主流NoSQL技术和应用，有大规模系统设计和开发经验者优先；
5. 熟悉常用的设计模式；有较强的架构能力和良好代码规范；
6. 熟悉常见的中间件、分布式解决方案及其原理：分布式缓存、SOA、消息中间件，负载均衡、连接池等；
7. 熟悉容器、K8S、ServiceMesh相关技术；
8. 有较强的分析和解决复杂问题的能力，善于捕捉业务需求、架构设计、编码中存在的问题，并给出有效的解决措施和方法；学习能力强，对新事物保持好奇心，有良好的沟通能力和团队协同能力。

## 岗位职责
1. 负责AI应用算法服务的开发和维护，完成从需求到设计、开发和上线等整个项目周期内的工作；
2. 负责构建高可靠性、高可用性、高可扩展性的分布式AI大模型与AI Agent服务，满足日趋复杂的业务需求。
